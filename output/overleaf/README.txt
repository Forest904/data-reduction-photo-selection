Overleaf export for the Data Reduction report.

1. Upload this directory without changing its structure.
2. Set main.tex as the main document.
3. Compile with pdfLaTeX.

docs/report/report.tex is the canonical source in the repository. main.tex
contains the same report text, except that its graphic path is "figures/" so
the export remains self-contained. When the canonical report or generated
figures change, refresh main.tex, references.bib, and all six PNG files here.
